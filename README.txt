M5ATOM_Lchika-TX.ino
送信機側のスケッチ

12行目に、受信機側のMACアドレスを記載してください。
受信機側のスケッチでシリアルモニターにアドレスを抽出できます。
uint8_t slaveAddress[] = { 0x**, 0x**, 0x**, 0x**, 0x**, 0x** }; // 受信機のMACアドレスに書き換えます。

9行目に、明暗の閾値を設定します。実使用環境でのCdSセルの出力値より最適値を判断して設定します。
const int CDS_VAL = 0;　照明ON/OFF時の明暗差が明確な場所では「0」で大丈夫なようです。


M5ATOM_Lchika-RX.ino
受信機側のスケッチ

受信機側のスケッチでシリアルモニターにアドレスを抽出できます。
18行目から25行目に　LED Matrixで表示する文字・絵を作成します。

受信機側は（送信機側も）電源接続後は一度リセットしたほうが良いです。
